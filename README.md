# 📝 Flask Note-Taking API with Grammar Checker

This is a simple Flask-based backend API for a markdown note-taking app. It allows users to:
- Submit notes in markdown format
- Automatically check for grammar issues using LanguageTool
- Save valid notes
- Render saved markdown notes as HTML
- List all saved notes

---

## 🚀 Features

- ✅ Grammar checking using `language-tool-python`
- ✅ Markdown rendering using `markdown` module
- ✅ Save notes as `.md` files
- ✅ RESTful API endpoints for interaction

---

## 📦 Requirements

- Python 
- Flask
- markdown
- language-tool-python
